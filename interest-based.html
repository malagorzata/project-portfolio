BASIC WEB

INTEREST-BASED WEBSITE

Programs/languages used: HTML, CSS, Photoshop, Illustrator

TASK

TASK

The main task of this project was to create fully responsive website of the interest of my choice. This was supposed to be supplemented with UX design and full design documentation.

PROCESS

PROCESS

The process started with a moodboard that has allowed me to explore different style options. After choosing strong colours and material style for my website, I decided to add only B&W photos to ensure contrast against the bold background. Material style felt perfect to ensure that photos stand out. The website layout and design present strong utilisation of Contrast and Proximity laws in line with Gestalt Laws. To ensure that website is clear and readible for the user, trunk and 5sec tests were conducted. This has higlighted the issue of inadequate header colour that was changed as a result.

RESULT

OUTCOME

The project has allowed me to implement HTML and CSS knowledge gained throughout this module. I improved my usage of grid, flexbox and design laws. Additionially, adding testing to the process helped to improve the user experience. Additionially, and most importantly, this assignment has taught me creating responsive web design that is crucial for the user. Through understanding web design process better - from creating wireframes and UX design to responsive coding while utilising design law, I have learnt to think more holistically about planning the website.
