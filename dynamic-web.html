DYNAMIC WEB

FILM PHOTOGRAPHY WEBSITE

Programs/languages used:
HTML, CSS, JavaScript with API key, fetch and template, Github, Figma, XD, VSCode, Restdb.io, Illustrator, Photoshop

TASK

TASK

This module's task was to create a database on restdb.io and then fetch data with JavaScript by using API to display it on the website. This required using grid to make it responsive to enable the website to host infinite amount of products. Additionally, it involved design research and testing to maximise the result.

PROCESS

PROCESS

The project included prototyping the vision through Figma to understand the initial idea for the website. Additionally, the user research was conducted to correctly capture the essence of what we aimed to present. The result indicated that the retro colour scheme combined with more modern fonts is a good combination to attract our target audience.
